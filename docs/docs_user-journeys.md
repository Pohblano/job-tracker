# User Journeys
## Shop Visibility Board (SVB)

---

## Journey 1: Engineer (Viewer)

1. Walks into shop
2. Looks at TV
3. Instantly sees:
   - Active jobs
   - Status
   - Progress
4. No questions asked

Outcome: Reduced interruptions.

---

## Journey 2: David (Editor)

1. Receives triple-signed drawing
2. Opens admin UI
3. Creates job (30 seconds)
4. Job appears instantly on TV

Outcome: Immediate visibility.

---

## Journey 3: Shop Lead

1. Completes production batch
2. Updates pieces completed
3. Progress bar updates in real time

Outcome: Accurate live progress.

---

## Journey 4: Management

1. Passively views TV
2. Understands shop load and flow
3. No need to request reports

Outcome: Transparency without overhead.
